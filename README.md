# angelvazqr.github.io

### Setup
On terminal
```bash
git clone https://github.com/angelvazqr/angelvazqr.github.io.git
cd angelvazqr.github.io
```
- Install [jekyll](https://jekyllrb.com)
- Install [jekyll Sitemap](https://github.com/jekyll/jekyll-sitemap)

### Start
On terminal
```bash
jekyll server
```
